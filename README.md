# 🧠 Deep Learning - RNNs (LSTM & GRU)

This repository contains implementation and explanation of Recurrent Neural Networks (RNNs), including LSTM and GRU models.

## 📌 Project Overview

This project helps in understanding:
- Basic RNN concepts
- Vanishing gradient problem
- LSTM (Long Short-Term Memory)
- GRU (Gated Recurrent Unit)
- Sequence modeling

---

## 📂 Folder Structure

- `notebooks/` → Jupyter notebooks for learning & experiments  
- `src/` → Python scripts for models  
- `data/` → Dataset used  
- `models/` → Saved trained models  
- `outputs/` → Graphs & results  
- `docs/` → Theory notes
